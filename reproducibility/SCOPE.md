# Repository Scope

Included by design:

- Latest RSE submission package for Paper58.
- Original Paper58 LaTeX/PDF and earlier docx versions.
- Original Paper 5 world model manuscript.
- Paper8 experimental code, cached embeddings, results, and model checkpoints.
- Prithvi/AlphaEarth encoder ablation materials.
- ADK world-model implementation and generated world-model figures.
- Legacy runtime files imported by the Paper58 scripts.
- Original archive bundles used during Colab experiments.

Excluded by design:

- Python bytecode and `__pycache__` directories.
- Full unrelated ADK or AlphaEarth-System history/worktrees.
- Private environment files and credentials.

The goal is to preserve all content needed to inspect and reproduce Paper58, without importing unrelated product, UI, or historical worktree material.
